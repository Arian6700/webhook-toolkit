🌐 Webhook Toolkit
A multi-functional, high-performance toolkit made in python for Discord webhook operations (Educational Purpose Only).

⚠️ IMPORTANT DISCLAIMER
THIS TOOL IS STRICTLY FOR EDUCATIONAL PURPOSES ONLY.

✅ Use only on webhooks you own
✅ Use only in your own servers for testing
✅ Learn about API interactions and rate limiting
❌ NOT for spamming, raiding, or harassing others
❌ NOT for violating Discord's Terms of Service
❌ NOT for any illegal activities

✨ Features

⚡ Super Fast	Optimized multi-threaded operations for rapid execution
🔌 Proxy Support	HTTP/HTTPS/SOCKS proxy support for distributed requests
🛠️ Webhook Creator	Create webhooks in Discord channels easily
🗑️ Webhook Deleter	Delete existing webhooks with confirmation
💥 Webhook Spammer	⚠️ Educational Only - Demonstrate rate limiting effects
ℹ️ Webhook Info	Fetch and display detailed webhook information
📊 Mass Operations	Handle multiple webhooks simultaneously

🛠️ Installation
Python 3.8 or higher

Steps
bash
# Clone the repository
git clone https://github.com/Arian6700/webhook-toolkit
